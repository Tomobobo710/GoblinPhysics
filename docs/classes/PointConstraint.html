PointConstraint Class

Defined in: src\classes\Constraints\PointConstraint.js:1
Module: Goblin

Constrains a point on one body (or one body and a fixed world point) to coincide with a point on another - a ball-and-socket joint. Three rows lock relative linear velocity at the pivot; no rotational constraint, so both bodies remain free to rotate about the shared point.
