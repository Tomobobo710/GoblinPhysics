Constraint Class

Defined in: src\classes\Constraints\Constraint.js:1
Module: Goblin

Base class for velocity constraints solved by the IterativeSolver. Not used directly - concrete constraints (ContactConstraint, FrictionConstraint, HingeConstraint, PointConstraint, SliderConstraint, WeldConstraint) extend this via Object.create( Goblin.Constraint.prototype ) and populate rows with the ConstraintRow(s) that express their particular restriction.
